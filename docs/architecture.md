# Архитектура MVP

## Выбранный стек

- Язык: Python
- Telegram framework: `aiogram 3`
- ORM: `SQLAlchemy 2`
- База данных: SQLite

## Почему этот стек

- `aiogram` хорошо подходит для Telegram-ботов и удобно работает с командами, callback-кнопками и роутерами.
- `SQLAlchemy` дает понятную модель данных и позволит позже перейти с SQLite на PostgreSQL почти без изменений бизнес-логики.
- SQLite достаточно для MVP и локальной демонстрации.

## Компоненты

1. `src/main.py`
   Точка входа, инициализация бота, диспетчера и запуск polling.

2. `src/bot/config.py`
   Загрузка токена и строки подключения к БД из переменных окружения.

3. `src/bot/database/`
   Настройка SQLAlchemy, базовый класс моделей и модели таблиц.

4. `src/bot/services/users.py`
   Логика поиска и создания пользователя.

5. `src/bot/handlers/`
   Обработчики команд и callback-нажатий.

6. `src/bot/keyboards/`
   Inline-клавиатуры для меню и навигации.
